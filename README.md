# 🏥 Aculife Healthcare & Research — Flutter Web App

> A professional healthcare web application built with **Flutter Web**, deployed for a real client and actively used in production.

🌐 **Live Website:** [aculifehealthcareandresearch.com](https://aculifehealthcareandresearch.com)

---

## 📋 About The Project

This is a fully functional healthcare website developed for **Aculife Healthcare & Research** — a real client project built with Flutter Web and deployed to production. The website allows patients to explore services, view doctors, and book appointments online.

---

## ✨ Features

- 📅 **Appointment Booking** — Patients can book appointments directly through the website
- 👨‍⚕️ **Doctor List** — Browse available doctors with their details
- 🏥 **Services** — View all available healthcare services
- 📍 **Location** — Find the clinic location easily
- ℹ️ **About** — Information about the healthcare center

## 🚀 Getting Started

### Prerequisites
- Flutter SDK (3.0+)
- Dart SDK

### Installation

```bash
# Clone the repository
git clone https://github.com/Radoan-Ahmed/website.git

# Navigate to project directory
cd website

# Install dependencies
flutter pub get

# Run on web
flutter run -d chrome
```

### Build for Production

```bash
flutter build web
```
